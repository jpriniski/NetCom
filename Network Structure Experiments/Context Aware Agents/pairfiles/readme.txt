Name convention:
filename = ['connection_' network size(# of nodes) '.' total number of trials in the experiment '.' neighborsize '_' randseed];


csv files: include local connection info. Cols 1 & 2: node index of a connected pair; Col 3: trial number (e.g., from 1 to 40 trial in an experiment)

gif files: trial-by-trial animations showing the connections for the three runs with the first randseed numbers 1-3.

gif files: network connection figure over trials for the three runs with the first randseed numbers 1-3.
